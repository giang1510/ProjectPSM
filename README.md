## Getting Started

### Prerequisites

Install or update these software to the latest version

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/giang1510/ProjectPSM.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
